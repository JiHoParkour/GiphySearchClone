# GiphySearchClone

Giphy앱 Search 화면의 기능 셋을 유사하게 구현한 앱입니다.

구현 사항
1. giphy api를 이용한 키워드 검색
2. gif 검색 결과(이미지) 리스트 화면
4. 다양한 높이의 셀을 가진 콜렉션뷰
5. 이미지 캐시매니저로 한 번 불러온 이미지는 캐시에서 가져오기
6. 검색한 gif 상세 화면
7. gif favorite 여부 로컬 저장
